hubot-goo-hiragana
==================

gooラボAPIを使って日本語文字列をひらがなに変換する hubot-script

## Configuration

**HUBOT_GOO_API_TOKEN**

gooラボAPIのアプリケーションID

## Installation

1. `npm install hubot-goo-hiragana --save`
2. "hubot-goo-hiragana" を external-scripts.json に追加
4. Reboot Hubot

## Commands

```
hubot これ読んで <query> - <query>を読み上げます
```
